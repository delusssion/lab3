# Лабораторная работа №3: структуры данных и сортировки

## Цель проекта
- Реализовать и сравнить несколько алгоритмов сортировки.
- Предоставить интерактивное меню для работы с последовательностями, сортировками, стеком и бенчмарками.
- Продемонстрировать обработку ошибок и предельных условий для учебных задач.

## Используемые технологии
- Python 3.11+
- pytest для тестирования
- colorama (окраска вывода)
- uv/pip для управления зависимостями и виртуальным окружением

## Реализованное
- Сортировки: пузырьковая, быстрая, подсчетом, поразрядная, пирамидальная, ведерная.
- Генераторы тестовых массивов: случайные, почти отсортированные, с дубликатами, обратные, вещественные.
- Три реализации стека: на связном списке, на списке и на очередях.
- Последовательности: факториал и числа Фибоначчи (итеративно и рекурсивно с кэшированием).
- Бенчмарки с сопоставлением времени работы алгоритмов и печатью табличных результатов.

## Принятые решения и допущения
- Подсчетом и поразрядная сортировки работают только с неотрицательными числами; верхние границы задаются в `src/constants.py`.
- Ведерная сортировка ожидает данные в диапазоне `[0, 1]`; при других диапазонах выполняется нормализация.
- При отсутствии colorama весь вывод остается, но без цветового оформления.
- Бенчмарки перехватывают ошибки (RecursionError/MemoryError/ValueError), помечая неудачные прогоны как `N/A`.

## Тестирование
- **76 модульных тестов** с полным покрытием функциональности.
- Покрыты генераторы данных, сортировки, стек, последовательности и бенчмарки.
- Проверяются корректные результаты, граничные условия и обработка ошибок.

## Запуск тестов и программы
1. Создать и активировать виртуальное окружение:
```
python3 -m venv venv
source venv/bin/activate # macOS/Linux
```
ИЛИ
```
venv\Scripts\activate #Windows
```

2. Установить зависимости:
```
pip install -r requirements.txt
```
3. Запуск тестов
```
python3 -m pytest
```
4. Запуск программы
```
python -m src.main
```
ИЛИ
```
python src/main.py
```
